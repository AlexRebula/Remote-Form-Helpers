# README

This ColdFusion on Wheels plugin will enable Remote Form Helpers (jQuery) in your applications.

## Methods added

-	pageInsertHTML
-	pageReplaceHTML
-	pageRemove
-	pageShow
-	pageHide
-	renderRemotePage
-	renderJavascript
-	startRemoteFormTag
-	endRemoteFormTag
-	remoteLinkTo

## Callbacks supported

-	onSuccess
-	onError
-	onComplete
-	onBeforeSend
